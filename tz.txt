какие контейнеры лейаута могут быть? контейнер лейаута - это одна из ячеек. 
контейнеры лейаута - по умолчанию это элементы находящиеся один под другим
так же контейнеры служат для позицианирования абсолютных блоков относительно их верхнего левого края
- блочный глобальный
- блочный глобальный абсолютный
- блочный абсолютный
- блочный неглобальный с контейнерами колонок
- контент модуля

какие типы лейаута потребуются для site draw?
- блочный глобальный. для хедера и футера, что бы потом они могли использоваться и на др. страницах
- блочный глобальный абсолютный. для контейнера главной страницы

* контейнеры (html) лейаута должны создаваться динамически. из данных

о лейаутах разделах и модулях
-----------------------------
привязывается тип контейнера лейауты к разделу или к модуля пока не ясно, потому как разделы используются
только лишь для построения дерева меню. в котором каждый из модулей будет становится листиком
